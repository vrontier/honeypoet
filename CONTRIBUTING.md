# Contributing

The Honeypo(e)t is an art project first, software second.

## Feature Requests

You're welcome to open an issue with ideas or suggestions. That said, this is a creative project with a specific artistic vision — we can't promise we'll take every request on board, and many won't fit the direction we're headed.

No hard feelings either way.

## Want to Take It in a Different Direction?

Fork it. The code is Apache 2.0 licensed and you're free to build your own version. We'd love to see what you make.

## Bug Reports

Genuine bugs — things that are broken, not things that are different from what you'd build — are always welcome. Open an issue with enough detail to reproduce it.
